Lista de Bebidas

Explorando o Mundo das Bebidas com Vue 3, Node.js e MongoDB Atlas

Este projeto é o resultado de um período dedicado ao aprendizado, com o objetivo de criar uma aplicação completa para gerenciamento de uma lista de bebidas. A aplicação abrange tanto o desenvolvimento do frontend com Vue 3 e Vuetify 3 quanto a integração de um backend em Node.js que se comunica com um banco de dados MongoDB Atlas hospedado na nuvem.

Funcionalidades:

  • Armazenar Detalhes:Pode armazenar descrição e imagem do produto.
  • Remover Bebidas: Exclua bebidas da lista quando necessário.
  • Atualizar Detalhes: Pode alterar descrições e imagens dos produtos.
  • Deletar Bebidas:Remova bebidas permanentemente da lista.

    • Este projeto representa um estudo sólido de operações CRUD (Criar, Ler, Atualizar, Excluir) em Vue 3, com armazenamento de dados em um banco de dados MongoDB na nuvem. Ele também inclui recursos aprimorados, como a exibição de imagens de produtos.

    Média de Idades e Gêneros

    Análise de Dados Simplificada com Node.js e Bootstrap

    Este projeto representa o resultado de um período de aprendizado dedicado à criação de uma API básica em Node.js, com uso de requisições HTTP do tipo POST e GET. A API é hospedada localmente (localhost) e segue o padrão MVC (Model-View-Controller). A rota criada é capaz de percorrer um array de números, calcular quantidades e médias, e fornecer os resultados ao cliente em formato JSON.

    Funcionalidades:

  • Adicionar Pessoas:Você pode adicionar informações sobre pessoas, incluindo NOME, IDADE e SEXO..
  • Exibir Informações:O projeto fornece informações essenciais, incluindo a quantidade de homens, a quantidade de mulheres, a média de idade dos homens e a média de idade das mulheres.

    • Este projeto oferece uma abordagem prática para coletar e analisar dados, e foi projetado para simplificar a visualização de informações-chave.
